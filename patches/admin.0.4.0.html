Tot ! Patches

Tot ! Admin 0.4.0->0.4.1 (Alpha)

New

  • 0.4.0 Economy Update - The Duke Nukem Forever of Tot ! Mods
    • Trader definition for puppets - Create a trader definition and assign it to one or more puppets. Buy/Sell multiplier allow you to fine tune the pricing at various level
    • Trading UI - Trade with NPCs but also with players using a Divinity 2 inspired UI. Open the action wheel on a player character, and open an exchange window for trading goods. No more dumping your shit on the ground like savages or running after your payment rolling down the hill.
    • NPC Trader - Craftable by players, this barebone NPC can hold plenty of goods to sell, but also buy some.
    • Non farmable coins - 19 different coin items to use in various ways, with no weight and max stack of 1,000,000.
  • 0.4.0 A radial menu button to hide station thralls
  • 0.4.0 Admin Flags - Added a new window for accessing admin flags, that can be accessed using /flags command.
  • 0.4.0 Visual Clue permission was moved to "Tot ! Admin - Flags" category
  • 0.4.0 Ignore Kit Restriction permission was moved to "Tot ! Admin - Flags" category and is now an Admin Flag
  • 0.4.0 Ignore Warp Restriction permission was moved to "Tot ! Admin - Flags" category and is now an Admin Flag

Changes/Improvements

  • 0.4.0 Items in kits can now have a max stack size beyond the item max stack size. This will generate multiple stack when redeeming the kit.
  • 0.4.0 Kit redeem feedback has been improved.
  • 0.4.0 Attribute value in the attribute widget editor can now be directly typed in.
  • 0.4.0 /summon now has a dedicated permission
  • 0.4.1 Adding permissions for trading with a player and hidding a station thrall
  • 0.4.1 NPC trader for players now ignore land claim
  • 0.4.1 Opening a trade window is now only possible while the target is standing passively
  • 0.4.1 When trading with another player, the other side is now hidden

Fixes

  • 0.4.1 Vanilla armor customization on puppets/npcs did not work properly
  • 0.4.1 Puppet trader were enable to spawn items in some situation
  • 0.4.1 Show station thralls had a clipping nacked body
  • 0.4.1 Hide/Show station thrall button was showing on things it should not
  • 0.4.1 Adding a horizontal scroll bar in the trader definition screen
  • 0.4.1 Trading was reseting constantly in some cases
